# Meu Portfólio de Data Science & AI

Bem-vindo(a) ao meu laboratório de dados! Este repositório funciona como um hub central que reúne meus projetos, estudos de caso e experimentos nas áreas de Data Science, Machine Learning, Inteligência Artificial e desenvolvimento de soluções baseadas em IA.

---

## Sobre o Portfólio

Este portfólio representa minha evolução prática e técnica, com foco em construir soluções reais e aplicáveis. Aqui você encontrará projetos nas seguintes áreas:

### Data Science & Análise de Dados

* **Manipulação e Limpeza de Dados:** Uso de **Pandas** para transformação, tratamento e preparação de dados.
* **Visualização de Dados:** Criação de gráficos e dashboards com **Matplotlib**, **Seaborn** e **Plotly**.
* **Análise Exploratória (EDA):** Identificação de padrões, correlações e insights relevantes.
* **Estatística Aplicada:** Uso de fundamentos estatísticos para tomada de decisão baseada em dados.
